How does ThermoState Calculate?

ThermoState uses CoolProp Library to calculate properties. It provides a GUI for the library with some added convenient features. It doesn't support fluid mixtures yet.

Visit CoolProp Website

Helmholtz energy formulations

Calculations are based on Helmholtz energy formulations. ThermoState uses HEOS Backend from CoolProp.
