Features

Thermophysical Properties

Calculate properties like Entropy, Enthalpy, Prandtl number, Viscosity

State Diagrams

Automatically plots calculated states on T-s, P-h and h-s diagrams

Flow

You can model cycle by defining a chain of related states.

History

Automatically saves your calculations for future sessions.

SI & English Units

You can set your preferred unit system or, customize unit per Quantity

How does ThermoState Calculate?

ThermoState uses CoolProp Library to calculate properties. It provides a User Interface for the library with some added convenient features. It doesn't support defining fluid mixtures yet. You don't need to be online to use it.

  • Helmholtz energy formulations

    ThermoState uses HEOS Backend from CoolProp.

  • 100+ Fluids

    A lot of fluids are available including Air, Water, R134a.

  • 10+ Input combinations

    Multiple input combinations are available for specifying a state.

Useful Links

  • CoolProp

    A C++ library that implements Pure and pseudo-pure fluid equations of state and transport properties for 122 components

  • REFPROP

    NIST Reference Fluid Thermodynamic and Transport Properties Database. Based on the most accurate pure fluid and mixture models currently available.

  • SmoWeb

    An open Computation platform that includes online fluid properties calculator for liquid, vapor and super-critical states based on CoolProp

  • ThermoCycle

    An open-source library for dynamic modelling of thermal systems developed in the Modelica language.

  • ACHP

    ACHP is an open source program written to model direct expansion and secondary loops for residential-scale systems.
