Features

Thermophysical properties

Calculate properties like Entropy, Enthalpy, Prandtl number, Viscosity etc.

State Diagrams

Automatically plots calculated states on TS, Ph and hs diagrams

Flow

You can model cycle by defining a chain of related states.

History

Automatically saves your calculations for future sessions. Continue where you left off.

SI & English Units

You can set your preferred unit system or, customize unit per Quantity

How does ThermoState Calculate?

ThermoState uses CoolProp Library to calculate properties. It provides a User Interface for the library with some added convenient features. It doesn't support defining fluid mixtures yet.

Visit CoolProp Website

  • Helmholtz energy formulations

    ThermoState uses HEOS Backend from CoolProp.

  • 100+ Fluids

    A lot of fluids are available including Air, Water, R134a.

  • 10+ Input combinations

    Multiple input combinations are available for specifying a state.
